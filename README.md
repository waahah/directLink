# directLink

>主流博客网站站外链接直达。

>与其他脚本区别：
>1. 其他脚本只在页面响应成功后处理一次链接，未考虑界面后续ajax或者iframe加载产生dom变化从而出现新的链接
>2. 或者采用定时器定时刷新所产生较多电脑不必要的负载 

>由于在时间线拉到下面动态获取新的问答或者展开评论的时候，Dom 会变化，所以此脚本动态监测页面dom变化，dom结构放生变化后才会遍历检查替换所有新出现的相关链接。
